Brightcove Open Source-Projekte

Unten finden Sie eine Liste aller aktuellen Open Source-Projekte für Brightcove-Technologien, gruppiert nach Kategorien.
Open Source-Projekte werden von Brightcove nicht offiziell unterstützt.

Willkommen bei Brightcove Open Source. Hier finden Sie eine Reihe von Projekten, die von Brightcove-Mitarbeitern oder Mitgliedern der Brightcove-Community erstellt wurden, um Ihre Brightcove-Entwicklung zu beschleunigen.

Wenn Sie daran interessiert sind, zu Projekten beizutragen, besuchen Sie die einzelnen Projektstandorte auf Github. Wenn Sie ein Projekt haben, das Sie hier auflisten möchten, kontaktiere uns.

Aktuelle APIs

  1. Fehlerbehebungsskript für API-Anforderungen

    Fehlerbehebung bei HTTP-Anforderungen und -Antworten auf / von Brightcove-APIs.

    https://github.com/BrightcoveOS/api-request-troubleshooter-script

  2. Brightcove-Uploader

    Frontend-Modul für den direkten mehrteiligen Upload auf s3 mit geheimer Sauce für die Brightcove-Aufnahme.

    https://github.com/BrightcoveOS/evaporate-brightcove

  3. Brightcove Playback API Wrapper

    Dies ist ein PHP-Wrapper für die Brightcove-Wiedergabe-API geschrieben von Theresa Newman.

    https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

  4. PHP Wrapper für Brightcove Dynamic Ingest

    Dieses Projekt bietet einen Ausgangspunkt für die Integration von Brightcove Dynamische Aufnahme-API in Ihre Bewerbung. Es bietet einfache Möglichkeiten zum Hinzufügen und Aktualisieren von Videos und zugehörigen Medien mithilfe von Pull-basierter Aufnahme oder Hochladen von Quelldateien.

    https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

  5. API-Fassade

    Diese bescheidene Bibliothek soll eine einfache Fassade über die Server-APIs von Brightcove sein. Da der Mob zusätzliche Funktionen benötigt, werden diese hier hinzugefügt.

    https://github.com/nwbb/node-brightcove

  6. Player Management API-Client

    Mit diesem Modul können Sie auf die Brightcove Player-Verwaltungs-API zugreifen. Dokumentation für die zugrunde liegende API finden Sie Hier.

    https://github.com/BrightcoveLearning/Player-Management-Client

  7. Beispiele für Brightcove-APIs mit CSharp

    Dieses Projekt zeigt, wie die neuen Brightcove-APIs mit C # verwendet werden können.

    Bitte beachten Sie, dass dies in Desktop-Umgebungen (getestet) funktioniert, aber wahrscheinlich nicht funktioniert, wenn es aufgrund von CORS in ASP.NET-Seiten verwendet wird. Wenn Sie auf dieses Problem stoßen, ist die einzige Möglichkeit, dies zu lösen, die Verwendung des von Brightcove bereitgestellten PHP-Proxys in allen ihren CMS-API Beispielseiten.

    Der gesamte Code wird wie besehen und nur als Beispiel bereitgestellt. Verwenden Sie es auf eigenes Risiko. Es wird keine Unterstützung in irgendeiner Weise, Form oder Gestalt bereitgestellt.

    https://github.com/houseofmackee/BrightcoveCS

  8. Brightcove-APIs Python-Tools

    Dieses Repository enthält in Python geschriebene Tools und Module, die sich mit Brightcove-bezogenen APIs befassen und Aufgaben wie das Durchlaufen der gesamten Videobibliothek und das Ausführen von Aktionen für die Videos vereinfachen.

    Getestete Version (en) von Python: 3.8.3

    Der gesamte Code wird wie besehen und nur als Beispiel bereitgestellt. Verwenden Sie es auf eigenes Risiko. Es wird keine Unterstützung in irgendeiner Weise, Form oder Gestalt bereitgestellt.

    https://github.com/houseofmackee/BrightcovePY

  9. Player Management API-Client

    Mit diesem Modul können Sie auf die Brightcove Player-Verwaltungs-API zugreifen. Dokumentation für die zugrunde liegende API finden Sie Hier.

    https://github.com/BrightcoveLearning/Player-Management-Client

Player und SDKs

  1. Video.js Plugin Framework Generator

    Dies ist ein eigenwilliger Yeoman-Generator für video.js-Plugins. Es basiert auf den Empfehlungen des Kernteams von video.j sowie auf Tools und Konventionen für die Entwicklung von Plugins bei Brightcove.

    https://github.com/videojs/generator-videojs-plugin

Systemintegrationen

  1. Brightcove Video Connect für SharePoint 2013

    Mit Brightcove Video Connect für SharePoint 2013 können Publisher Videos und Video-Wiedergabelisten direkt aus SharePoint hinzufügen, durchsuchen und verwalten. Die Player-Komponente ermöglicht die einfache Integration von Brightcove Video Cloud-Assets direkt in SharePoint-Seiten.

    http://go.brightcove.com/bc-video-connect-sharepoint2013

  2. Brightcove Video Connect für Drupal

    Brightcove Video Cloud ist eine Video-Hosting- und Publishing-Lösung für die Bereitstellung von Videos in professioneller Qualität für digitale Zielgruppen auf jedem Gerät. Es bietet hochgradig anpassbare Player, HD-Streaming, Synchronisierung mit CDNs von Drittanbietern und mit YouTube. Mit Brightcove Video Connect für Drupal können Sie Videoinhalte auf Ihren Drupal-basierten Websites effizient hinzufügen und verwalten.

    https://www.drupal.org/project/brightcove

  3. Brightcove Video Connect für WordPress

    Möchten Sie Ihre Brightcove-Video- und Wiedergabelistenbibliothek nativ in WordPress verwalten? Mit diesem von 10up.com entwickelten Plugin können Sie mehrere Konten und Videobibliotheken verwalten, Videos hochladen und zu Wiedergabelisten hinzufügen, Shortcodes mit Ihren Videos rendern und alles über die WordPress-Administrationsoberfläche.

    https://wordpress.org/plugins/brightcove-video-connect/

  4. SharePoint 2013 Connector

    Bietet die Integration zwischen SharePoint 2013 und mehreren Video Cloud-Konten.

    https://github.com/brightcove/sharepoint-connector

  5. Brightcove Video Connect für Adobe Experience Manager

    Brightcove Video Connect für Adobe Experience Manager (AEM) ist die einfachste Möglichkeit, den Workflow zwischen Brightcove Video Cloud und AEM zu integrieren. Mit Brightcove Video Connect können AEM-Benutzer Videos direkt über die AEM-Benutzeroberfläche hinzufügen, durchsuchen und verwalten.

    http://go.brightcove.com/bc-video-connect-aem

  6. CQ5-Komponente

    Die CQ5-Komponente integriert Brightcove

    Technologien: CQ5

    https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

  7. SalesGabelstapler

    Salesforce bietet eine Massen-API, um die massive Datensynchronisation von einem Datenspeicher zu Salesforce zu vereinfachen. Dieses Projekt bietet ein Juwel für die Verwendung der Salesforce Bulk-API in einer Ruby-App.

    Technologien: Salesforce, Ruby

    https://www.github.com/brightcove/salesforklift

  3. LG Beispiel-App

    Dies ist eine LG-Videoanwendung, die mehrere Wiedergabelisten eines Brightcove-Kontos anzeigt. Die Anwendung wurde so konzipiert, dass nur eine minimale Konfiguration erforderlich ist, und ermöglicht es Kunden, Anpassungen vorzunehmen, um die Benutzeroberfläche zu ändern oder Funktionen entsprechend den gewünschten Funktionen hinzuzufügen / zu entfernen.

    Technologien: LG, Fernseher

    https://www.github.com/BrightcoveOS/LG-Sample-App

    Technologien: Roku, Fernseher

  5. Java Commons

    Technologien: Java

  6. Diamant

    Diamond ist ein Python-Daemon, der Systemmetriken sammelt und in Graphite veröffentlicht. Es ist in der Lage, CPU-, Speicher-, Netzwerk-, E / A-, Lade- und Festplattenmetriken zu erfassen. Darüber hinaus bietet es eine API zum Implementieren von benutzerdefinierten Kollektoren zum Sammeln von Metriken aus nahezu jeder Quelle.

    Technologien: Graphit, Python

    https://github.com/python-diamond/Diamond
