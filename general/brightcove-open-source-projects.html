    Brightcove Open Source Projekte

    Nachfolgend finden Sie eine Liste aller aktuellen Open Source-Projekte für Brightcove-Technologien, gruppiert nach Kategorien.
    Open Source-Projekte werden von Brightcove nicht offiziell unterstützt.

    Willkommen bei Brightcove Open Source. Hier finden Sie eine Reihe von Projekten, die von Brightcove-Mitarbeitern oder Mitgliedern der Brightcove-Community erstellt wurden, um Ihre Brightcove-Entwicklung zu beschleunigen.

    Wenn Sie daran interessiert sind, zu Projekten beizutragen, besuchen Sie die einzelnen Projektstandorte auf Github. Wenn Sie ein Projekt haben, das Sie hier aufgelistet haben, kontaktieren Sie uns.

    Aktuelle APIs

    1. API Anfrage Fehlerbehebung Script

      Beheben Sie HTTP-Anfragen und -Antworten zu/von Brightcove-APIs.

      https://github.com/BrightcoveOS/api-request-troubleshooting-script

    2. Brightcove Uploader

      Frontend-Modul für den direkten mehrteiligen Upload auf S3 mit geheimer Sauce für Brightcove Intest.

      https://github.com/BrightcoveOS/evaporate-brightcove

    3. Brightcove Wiedergabe-API-Wrapper

      Dies ist ein PHP-Wrapper für die Brightcove Playback-API , geschrieben von Theresa Newman.

      https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

    4. PHP-Wrapper für Brightcove Dynamic Intest

      Dieses Projekt bietet einen Ausgangspunkt für die Integration von Brightcove Dynamic Ingest API in Ihre Anwendung. Es bietet einfache Möglichkeiten zum Hinzufügen und Aktualisieren von Videos und zugehörigen Medien mithilfe von Pull-basiertem Intest oder Hochladen von Quelldateien.

      https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

    5. API-Fassade

      Diese bescheidene Bibliothek soll eine einfache Fassade gegenüber den Server-APIs von Brightcove sein. Da der Mob zusätzliche Funktionalität verlangt, wird er hier hinzugefügt.

      https://github.com/nwbb/node-brightcove

    6. Spieler Management-API-Client

      Dieses Modul kann für den Zugriff auf die Brightcove Player Management API verwendet werden. Die Dokumentation für die zugrunde liegende API finden Sie hier.

      https://github.com/BrightcoveLearning/Player-Management-Client

    7. Brightcove APIs Beispiele zur Verwendung von CSharp

      Dieses Projekt zeigt, wie die neuen Brightcove-APIs mit C # verwendet werden können.

      Bitte beachten Sie, dass dies von Desktop-Umgebungen aus funktioniert (getestet), aber aufgrund von CORS wahrscheinlich nicht funktioniert, wenn es in ASP.NET-Seiten verwendet wird. Wenn Sie auf dieses Problem stoßen, glaube ich, dass die einzige Möglichkeit, dies zu lösen, darin besteht, den von Brightcove bereitgestellten PHP-Proxy auf allen ihren CMS API Beispielseiten zu verwenden.

      Der gesamte Code wird unverändert und nur als Beispiel bereitgestellt. Verwenden Sie es auf eigenes Risiko. Es wird in keiner Weise unterstützt, in Form oder Form.

      https://github.com/mikelau13/BrightcoveAPI

    8. Spieler Management-API-Client

      Dieses Modul kann für den Zugriff auf die Brightcove Player Management API verwendet werden. Die Dokumentation für die zugrunde liegende API finden Sie hier.

      https://github.com/BrightcoveLearning/Player-Management-Client

    Spieler und SDKs

    1. Video.js Plugin Framework Gener

      Dies ist ein eigensinnig Yeoman-Generator für video.js Plugins. Es basiert auf den Empfehlungen des Kernteams von video.js sowie auf Tools und Konventionen für die Entwicklung von Plugins bei Brightcove.

      https://github.com/videojs/generator-videojs-plugin

    Systemintegrationen

    1. Brightcove Video Connect für SharePoint 2013

      Der Brightcove Video Connect for SharePoint 2013 ermöglicht es Publishern, Videos und Video-Wiedergabelisten direkt von SharePoint aus hinzuzufügen, zu durchsuchen und zu verwalten. Seine Player-Komponente ermöglicht die einfache Integration von Brightcove Video Cloud-Ressourcen direkt in SharePoint-Seiten.

      http://go.brightcove.com/bc-video-connect-sharepoint2013

    2. Brightcove Video Connect für Drupal

      Brightcove Video Cloud ist eine Video-Hosting- und Publishing-Lösung zur Bereitstellung von Videos in professioneller Qualität für digitale Zielgruppen auf jedem Gerät. Es bietet hochgradig anpassbare Player, HD-Streaming, Synchronisierung mit CDNs von Drittanbietern und mit YouTube. Mit Brightcove Video Connect for Drupal können Sie Videoinhalte effizient auf Ihren Drupal-basierten Websites hinzufügen und verwalten.

      https://www.drupal.org/project/brightcove

    3. Brightcove Video Connect für WordPress

      Möchten Sie Ihre Brightcove Video- und Playlist-Bibliothek nativ aus WordPress heraus verwalten? Mit diesem von 10up.com entwickelten Plugin haben Sie die Möglichkeit, mehrere Konten und Videobibliotheken zu verwalten, Videos hochzuladen und zu Wiedergabelisten hinzuzufügen, Shortcodes mit Ihren Videos zu rendern und alles innerhalb der WordPress-Administrationsoberfläche.

      https://wordpress.org/plugins/brightcove-video-connect/

    4. SharePoint 2013-Anschluss

      Bietet die Integration zwischen SharePoint 2013 und mehreren Video Cloud-Konten.

      https://github.com/brightcove/sharepoint-connector

    5. Brightcove Video Connect für Adobe Experience Manager

      Brightcove Video Connect für Adobe Experience Manager (AEM) ist der einfachste Weg, den Workflow zwischen Brightcove Video Cloud und AEM zu integrieren. Mit Brightcove Video Connect können AEM-Benutzer Videos direkt über die AEM-Benutzeroberfläche hinzufügen, durchsuchen und verwalten.

      http://go.brightcove.com/bc-video-connect-aem

    6. CQ5-Komponente

      Die CQ5-Komponente integriert Brightcove

      Technologien: CQ5

      https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

    7. salesforklift

      Salesforce bietet Bulk-API, um die massive Datensynchronisierung von einem Datenspeicher zu Salesforce zu vereinfachen. Dieses Projekt bietet ein Juwel zur Verwendung der Salesforce Bulk API in einer Ruby-App.

      Technologien: Salesforce, Rubin

      https://www.github.com/brightcove/salesforklift

    Verschiedenes

    Diese Projekte passen in keine andere Kategorie.

    1. Perl Commons

      Dieses Projekt bietet allgemeine Klassen und Dienstprogramme für die Verwendung in anderen Perl-Projekten. Dies kann sowohl Bibliotheken von Drittanbietern als auch Brightcove Open Source-Bibliotheken umfassen.

      Technologien: Perl

      https://www.github.com/BrightcoveOS/Perl-Commons

    2. Metriken Graphit Pickle

      Eine Verlängerung auf Coda Hale

      Technologien: Graphit, Java

      https://www.github.com/BrightcoveOS/metrics-graphite-pickle

    3. LG Beispiel-App

      Dies ist eine LG-Videoanwendung, die mehrere Wiedergabelisten von einem Brightcove-Konto anzeigt. Die Anwendung wurde so konzipiert, dass eine minimale Konfiguration erforderlich ist, und ermöglicht es Kunden, Anpassungen vorzunehmen, um die Benutzeroberfläche zu ändern oder Funktionen entsprechend der von ihnen gewünschten Funktionalität hinzuzufügen/zu entfernen.

      Technologien: LG, TV

      https://www.github.com/BrightcoveOS/LG-Sample-App

    4. Roku Beispiel-App

      Diese Vorlage bietet einen Neuanfang für die Veröffentlichung Ihrer Brightcove-Inhalte auf Roku. Um mit der Veröffentlichung mit dieser Anwendung zu beginnen, benötigen Sie Folgendes:

      Technologien: Roku, Fernsehen

      https://www.github.com/BrightcoveOS/Roku-Sample-App

    5. Java-Commons

      Dieses Projekt bietet allgemeine Klassen und Dienstprogramme für die Verwendung in anderen Java-Projekten. Dies kann sowohl Bibliotheken von Drittanbietern als auch Brightcove Open Source-Bibliotheken umfassen.

      Technologien: Java

      https://www.github.com/BrightcoveOS/Java-Commons

    6. Diamant

      Diamond ist ein Python-Daemon, der Systemmetriken sammelt und in Graphite veröffentlicht. Es ist in der Lage, CPU-, Speicher-, Netzwerk-, E/A-, Lade- und Festplattenmetriken zu sammeln. Darüber hinaus bietet es eine API zum Implementieren benutzerdefinierter Collectors zum Sammeln von Metriken aus fast jeder Quelle.

      Technologien: Graphit, Python

      https://github.com/python-diamond/Diamond
