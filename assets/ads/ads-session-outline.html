***Anzeigeninformationen zu David Meeting***


Grundwortschatz

Arten von Werbeanzeigen

  • Lineare Anzeigen: Anzeigen, die während der Wiedergabe auftreten, nehmen Zeit in der Timeline in Anspruch (Beispiel ist ein Preroll)
  • Nichtlineare Werbung: Unterbricht die Wiedergabe nicht; Overlay- und Begleitanzeigen sind Beispiele

Anzeigen-Standards

  • VMAP: „Anzeigen-Playlist“; spezifiziert Preroll, Midroll (s) und Postroll (URL zu einem VMAP-Dokument); Format ist XML

    • DFP nennt VMAP-Dokumente als „Anzeigenregeln“; mit Anzeigenregeln können Sie alle Ihre Anzeigen auf dem Server konfigurieren, und jeder Kunde, der Anzeigenregeln versteht, verhält sich gleich; in „schlechten alten Zeiten“ Anzeigenkonfiguration am Client; Logik in das Publishing-System verschoben

  • VAST: Individuelle Anzeigendetails; VMAP verweist auf ein VASTINA-Dokument mit Informationen zur Anzeige

  • Beispiel VAST von der Registerkarte Netzwerk in Chrome Dev Tools

  • VPAID: Wie VAST, aber ein Standard für interaktive Anzeigen; Anzeige enthält ein Quiz- oder Goofy-Spiel (noch keine BC Player-Unterstützung)

Informationen zum IMA-Plu

  • IMA-Einschränkung: 2 Versionen der IMA-Bibliothek, Flash- oder HTML-Version können nicht beides verwenden

    • Standardeinstellungen für Plugin verwenden Flash, falls verfügbar (weil Flash-Anzeigen so weit verbreitet sind)
    • In der Konfiguration können Optionen für das Plugin standardmäßig auf HTML (AdtechOrder) umgestellt werden
    • Desktop Standard Flash, mobiler Standard-HTML-Code

  • Basisimplementierung von IMA Plugin

    • Fügen Sie JavaScript für das Plugin
    • IMA Plugin zum Spieler hinzufügen
    • ServerUrl für Anzeigen-Tag bereitstellen
    • Beispiel-Konfiguration

  • Dynamisches Ad-Targeting: Konfiguration des IMA-Plugins nach dem Sammeln von Kundeninformationen (z. B. Alter der Zuschauer, dann Anzeigenregel)

    • Fügen Sie JavaScript für das Plugin
    • Rufen Sie das IMA-Plugin auf und bestehen Sie an diesem
      • myPlayer.ima3 ({serverUrl:...})
  • Begleiter-Anzeigen: Das IMA-Plugin wird sich nahtlos mit linearen Anzeigen und Overlays befassen, Begleitanzeigen können schwierig sein, da die Kommunikation zwischen dem Player und dem Rest der Seite erfolgen muss.;; Verwenden Sie die Implementierung der In-Page-Einbettung
    • Verwenden Sie die Google Publisher Tags-Bibliothek (GPT) und füllen Sie automatisch die Begleiter-Werbeblöcke
    • Gutes Tutorial auf der IMA-Website
    • Verwenden Sie die In-Page-Einbettungsimplementierung für Begleitanzeigen, da iframe nicht mit der übergeordneten Seite interagieren
    • Beispiel: JS und HTML für Begleitwerbung
