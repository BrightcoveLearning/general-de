String-Encoder

Verwenden Sie zum URI-Codieren von Strings auf 3 Arten.

Fügen Sie die vorgeschlagenen Methodennamen hinzu. Diese sind im Allgemeinen nützlich, wenn Sie eine Zeichenfolge in eine URL einfügen müssen. Welches angemessen ist, hängt von der Situation ab. Nachfolgend finden Sie einige Beispiele, wie wir sie nützlich finden.

  • escape ist die älteste Methode und tut am wenigsten; Wir hatten keine Gelegenheit, es in unseren Apps zu verwenden
  • encodeURI ist im Allgemeinen für Zeichenfolgen ausreichend, die als Werte für URL-Parameter übergeben werden
  • encodeURIComponent ist der gründlichste Encoder, aber im Allgemeinen benötigen Sie ihn nur für Strings, die als Teil einer URL-Domäne/eines Pfades verwendet werden, oder wenn Sie eine URL als Parameter übergeben müssen (wir tun dies häufig, um API-Anfragen an Proxys zu übergeben über AJAX)
